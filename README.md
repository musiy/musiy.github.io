### useful commands
init new bundle (already have one)<br>
`bundle init`

install dependencies from gem file<br>
`bundle install --path vendor/bundle`

add new packet to existing bundle<br>
`bundle add jekyll`

serve site on localhost
`bundle exec jekyll serve`
